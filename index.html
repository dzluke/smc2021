Source Separation Methods for Computer-Assisted Orchestration

Luke Dzwonczyk, Léo Chédin, Alejandro Saldarriaga-Fuertes, Max Sherr, Hélène-Camille Crayencour, Carmine-Emanuele Cella

Below is a selection of targets and orchestrations. The full target orchestration is the result of orchestrating the target without performing any separation. The ground truth orchestration takes the four sources of the target and independently orchestrates them, representing an orchestration that could occur if source separation is performed perfectly.

Our main idea is to apply source separation methods on targets to create four sub-targets, then independently orchestrate the four sub-targets with different orchestras, and finally combine the orchestrations to play simultaneously. We then compare the separated solution to the orchestration of the same target with no separation performed.
